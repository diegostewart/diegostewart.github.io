# My Favorite Coding Tools

As a developer, having the right tools can make a huge difference in productivity and enjoyment. Here are some of my favorites.

## Code Editor: VS Code

- Lightweight and fast
- Extensive extension ecosystem
- Great Git integration
- Customizable themes

## Version Control: Git

- Distributed version control
- Branching and merging
- GitHub for collaboration

## Terminal: iTerm2 + Zsh

- Powerful shell
- Oh My Zsh for plugins
- Customizable prompts

## Package Managers

- npm for JavaScript
- pip for Python
- Homebrew for macOS

## Honorable Mentions

- Postman for API testing
- Figma for design
- Docker for containerization
